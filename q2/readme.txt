Commands:
make: will run code for first question 
make tweak: will run code for second question 
make valgrind: will run the code for question 2 with valgrind
make stab: will run code for third question 
make clean: removes executables and object files 

- To change the parameters change values in parameters.in or the command line arguments
- To change the function of beta to be optimized, change it in fun (line 100 of tweaking.f90)
- To change the method used for solving, change it in fun (line 98 of tweaking.f90)
- to change the compiler change line 1 of 'Makefile'
- To change the compilation flags change line 2 and 3 of Makefile
